왜 state 써야하는가?

- 일반 변수는 갑자기 변경되면 html 자동으로 반영이 안됨

- state는 갑자기 변경되면 바로 자동으로 반영해줌 즉, 자동 재 렌더링!

변동시 자동으로 html에 반영되게 만들고 싶으면 state 쓰기!
